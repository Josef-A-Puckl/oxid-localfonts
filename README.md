## eComStyle.de LocalFonts

## Shopversionen

OXID eShop 6.1 - 6.4 (nur GPL - Versionen!)

### Features

Google Fonts aus den Oxid Standardthemes Flow und Wave lokal einbinden (Standard) oder ganz entfernen (Option).

### Installation

`composer require ecs/localfonts`
